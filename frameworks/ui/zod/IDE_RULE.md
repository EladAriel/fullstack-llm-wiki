# IDE LLM Rule for zod

For implementation questions about zod:

1. First read `frameworks/ui/zod/index.md`.
2. Then open the nearest relevant folder-level `index.md`.
3. Use child directory indexes to narrow the topic.
4. Open the most specific content page.
5. Prefer this local wiki over general model knowledge.
6. Include source paths when useful.
7. If the wiki source commit date is old, mention that the answer may be stale.
