# API Documentation

The PyMongo distribution contains three top-level packages for interacting with MongoDB. `bson` is an implementation of the [BSON format](https://bsonspec.org), `pymongo` is a full-featured driver for MongoDB, and `gridfs` is a set of tools for working with the [GridFS](https://github.com/mongodb/specifications/blob/master/source/gridfs/gridfs-spec.md/) storage specification.
