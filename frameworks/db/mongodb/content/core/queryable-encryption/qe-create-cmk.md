# Create a {+cmk-long+}

## Overview

In this guide, you will learn how to generate a {+cmk-long+} in your {+kms-long+} of choice. Generate a {+cmk-long+} before creating your {+qe+}-enabled application.

> **Tip:** To learn more about the {+cmk-long+}, see
`qe-reference-keys-key-vaults`

## Before You Start

Complete the preceding tasks before continuing:

#. `Install a {+qe+} compatible driver and dependencies <qe-install>`

#. `Install and configure a {+qe+} library <qe-csfle-install-library>`

## Procedure

Select your key provider below.

## Next Steps

After installing drivers and dependencies and creating a {+cmk-long+}, you can `create your {+qe+} enabled application <qe-create-application>`.
