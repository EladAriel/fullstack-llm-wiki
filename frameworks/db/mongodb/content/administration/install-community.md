# Install MongoDB Community Edition

.. include:: /includes/minor-release.rst

Select your operating system to view installation instructions for MongoDB Community Edition. On certain operating systems, you can include or exclude MongoDB Search using the corresponding selector.

> **Note:** By default, MongoDB Community Edition does not include MongoDB Search. To use
MongoDB Search with MongoDB Community, you must install MongoDB Search
separately.
