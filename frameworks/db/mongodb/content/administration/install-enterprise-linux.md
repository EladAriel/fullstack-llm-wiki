# Install MongoDB Enterprise on Linux

.. include:: /includes/minor-release.rst

These documents provide instructions to install MongoDB Enterprise Edition for supported Linux systems.

`Install on Red Hat <install-mdb-enterprise-redhat-centos>` Install MongoDB Enterprise and required dependencies on Red Hat Enterprise or CentOS Systems using packages.

`Install on Ubuntu <install-mdb-enterprise-ubuntu>` Install MongoDB Enterprise and required dependencies on Ubuntu Linux Systems using packages.

`Install on Debian <install-mdb-enterprise-debian>` Install MongoDB Enterprise and required dependencies on Debian Linux Systems using packages.

`Install on SUSE <install-mdb-enterprise-suse>` Install MongoDB Enterprise and required dependencies on SUSE Enterprise Linux.

`Install on Amazon <install-mdb-enterprise-amazon-linux>` Install MongoDB Enterprise and required dependencies on Amazon Linux AMI.
