MongoDB supports draft 4 of JSON Schema, including [core specification](https://tools.ietf.org/html/draft-zyp-json-schema-04) and [validation specification](https://tools.ietf.org/html/draft-fge-json-schema-validation-00), with some differences. For details, see `jsonSchema-extension` and `json-schema-omission`.

For more information about JSON Schema, see the [official website](http://json-schema.org/)_.
