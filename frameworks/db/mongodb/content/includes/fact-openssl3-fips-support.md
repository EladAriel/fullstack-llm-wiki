Starting in MongoDB 6.0.7, FIPS mode supports [OpenSSL3](https://www.openssl.org/docs/man3.0/man7/migration_guide.html)_ for the following operating systems:

- Ubuntu 22.04
- RHEL 9
- Amazon Linux 2023
Starting in MongoDB 8.0, FIPS mode supports OpenSSL3 for Amazon Linux 2023.3.
