- If you `installed MongoDB <tutorial-installation>` with a package manager
such as `yum` or `apt` on Linux or `brew` on macOS, or with the MSI installer on Windows, a default `configuration file </reference/configuration-options>` has been provided as part of your installation:

- If you `installed MongoDB <tutorial-installation>`
