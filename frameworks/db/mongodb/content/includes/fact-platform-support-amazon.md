MongoDB {+version+} Community Edition supports the following :red:`64-bit` Amazon Linux release on `x86_64 <prod-notes-supported-platforms-x86_64>` architecture:

- Amazon Linux 2023
MongoDB only supports the 64-bit versions of this platform.

MongoDB {+version+} Community Edition on Amazon Linux also supports the `ARM64 <prod-notes-supported-platforms-ARM64>` architecture on select platforms.

See `prod-notes-supported-platforms` for more information.
