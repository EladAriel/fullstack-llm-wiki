Downgrade the `featureCompatibilityVersion` to |oldfcv|.

> **Important:** Verify you have removed all
`backward-incompatible features <8.3-downgrade-incompatible>`
before downgrading the binary version. Make sure to evaluate your
upgrade path before proceeding.

```javascript
db.adminCommand( { setFeatureCompatibilityVersion: "8.0", confirm: true } )
```

The :dbcommand:`setFeatureCompatibilityVersion` command performs writes to an internal system collection and is idempotent. If the command does not complete successfully, retry the command on the |target|.
