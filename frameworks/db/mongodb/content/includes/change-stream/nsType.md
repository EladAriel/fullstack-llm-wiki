Shows the namespace type of the collection created by this event.

`"collection"` Indicates the event creates a collection.

`"timeseries"` Indicates the event creates a timeseries collection.

`"view"` Indicates the event creates a non-timeseries view.

.. versionadded:: 8.1 (also available in 8.0.5).
