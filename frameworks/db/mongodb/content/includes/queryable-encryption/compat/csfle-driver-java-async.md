### Driver Compatibility

To use {+csfle+} with the :driver:`Java Reactive Streams </reactive-streams/>` driver, install driver version 1.12.0 or later. To use the Key Rotation API, install `mongodb-crypt` version {+mongodb-crypt-version+} or later.
