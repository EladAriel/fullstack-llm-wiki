**For supported Linux Operating Systems:** To install the Server package, follow the `install on Linux tutorial <install-enterprise-linux>` and install the `mongodb-enterprise` server package. Alternatively, specify `mongodb-enterprise-cryptd` instead to install only the `mongocryptd` binary. The package manager installs the binaries to a location in the system PATH.

**For OSX:** To install the Server package, follow the `install on MacOS tutorial <install-enterprise-macos>`. The package manager installs binaries to a location in the system PATH.

**For Windows:** To install the Server package, follow the `install on Windows tutorial <install-enterprise-windows>`. You must add the `mongocryptd` package to your system PATH after installation. Follow the documented best practices for your Windows installation to add the `mongocryptd` binary to the system PATH.

**To install from an official tarball / ZIP archive:** To install from an official archive, follow the documented best practices for your operating system to add the `mongocryptd` binary to your system PATH.
