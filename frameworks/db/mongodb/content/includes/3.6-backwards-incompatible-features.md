The following 3.6 features require that `featureCompatibilityVersion` be set to `"3.6"`:

- `UUID for collections <collections-uuids>`
- :query:`$jsonSchema` schema validation
- `/changeStreams`
- Chunk aware secondaries
- View definitions, document validators, and partial index filters that
use 3.6 query features

- Sessions and retryable writes
- Users and roles with `authenticationRestrictions <3.6-authentication-restrictions>`
