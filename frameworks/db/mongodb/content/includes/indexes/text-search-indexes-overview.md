For data hosted on MongoDB, you can support full-text search with {+fts+} indexes. To learn more, see :atlas:`Create a {+fts+} Index </atlas-search/create-index/>`.

For self-managed (non-Atlas) deployments, MongoDB provides a `text` index type that supports searching for string content in a collection. To learn more about self-managed text indexes, see `index-feature-text`.
