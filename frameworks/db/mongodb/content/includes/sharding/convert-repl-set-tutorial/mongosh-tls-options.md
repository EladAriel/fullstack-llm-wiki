If your deployment uses X.509 authentication, specify these :binary:`mongosh` options:

- :option:`--tls <mongosh --tls>`
- :option:`--tlsCAFile <mongosh --tlsCAFile>`
- :option:`--tlsCertificateKeyFile <mongosh --tlsCertificateKeyFile>`
