You can create a configuration file to specify options for MongoDB. To do so, create a file named `mongod.cfg` in the `\bin\` directory within the directory where you extracted the MongoDB files (`<install directory>\bin\mongod.cfg`). For more information on creating a config file, see `conf-file`.
