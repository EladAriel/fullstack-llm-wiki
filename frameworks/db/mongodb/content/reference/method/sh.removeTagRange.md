## Example

Given a shard key of `{state: 1, zip: 1}`, the following operation removes an existing tag range covering zip codes in New York State:

```javascript
sh.removeTagRange( "exampledb.collection",
                { state: "NY", zip: MinKey },
                { state: "NY", zip: MaxKey }
              )
```
