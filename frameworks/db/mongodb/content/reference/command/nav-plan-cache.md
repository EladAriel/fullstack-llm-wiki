# Query Plan Cache Commands

.. include:: /includes/extracts/commands-toc-explanation.rst

Starting in MongoDB 8.0, use query settings instead of `index filters <index-filters>`. Index filters are deprecated starting in MongoDB 8.0. To add query settings and explore examples, see :dbcommand:`setQuerySettings`.
