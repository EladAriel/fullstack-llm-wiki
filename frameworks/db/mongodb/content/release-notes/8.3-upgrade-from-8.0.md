# Upgrade |oldversion| to |newversion|

.. include:: /includes/8.3-upgrade-from-8.0-replacements.rst

.. include:: /includes/minor-release.rst

This page lists procedures used to upgrade from |oldversion| to |newversion|.

> **Important:** To upgrade to MongoDB |newversion| from a |oldversion| deployment,
the |oldversion| deployment must have `featureCompatibilityVersion`
set to |oldversion|. To check the version:
.. code-block:: javascript
   db.adminCommand( { getParameter: 1, featureCompatibilityVersion: 1 } )
