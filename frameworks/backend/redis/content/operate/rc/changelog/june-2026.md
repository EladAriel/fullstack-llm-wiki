## New features

### Billing Admin API access

Users with the [Billing Admin role]({{< relref "/operate/rc/security/access-control/access-management#team-management-roles" >}}) can now use the [Redis Cloud REST API]({{< relref "/operate/rc/api" >}}) to perform billing operations. See [Team management roles]({{< relref "/operate/rc/security/access-control/access-management#team-management-roles" >}}) for the full list of permissions for each role.

### Redis Data Integration 

[Redis Data Integration (RDI)]({{< relref "/operate/rc/rdi" >}}) is now generally available on Redis Cloud. RDI lets you create data pipelines that ingest data from a supported primary database into Redis, so your cache stays in sync with live data and is always ready for queries.

See [Data Integration]({{< relref "/operate/rc/rdi" >}}) to learn how to set up data pipelines with Redis Cloud.
